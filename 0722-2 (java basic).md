# Stack

**Stack class**

 - 바닥이 막힌 박스 형태 (접근 통로가 하나)
 - 후입 선출 : LIFO (Last In First Out)
 - 주요 메서드 : push(), pop(), peek()








# Queue

**Queue interface**

 - 양쪽이 뚫린 파이프 형태 (접근 통로가 둘)
 - Queue interface 구현 -> LinkedList class
 - 선입 선출 : FIFO (First In First Out)
 - 주요 메서드 : offer(), poll(), peek()
