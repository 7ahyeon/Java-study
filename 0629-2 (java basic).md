# Java-study

**자바 프로그램 개발 과정**

                           자바 소스 파일(.java)

                           -> javac 명령어 실행(compile)
                          
                          바이트 코드 파일(.class)
                          
                          -> java 명령어 실행
                          
                          JVM : 기계어 번역 후 실행

**인터프리터** 프로그래밍 언어의 소스 코드를 바로 실행하는 프로그램 또는 환경 (interpreter) <-> 컴파일러
