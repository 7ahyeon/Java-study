# Java-study

**클래스 Class**
- 구현체 : 기능 (method) 구현 ({} : 구현부)
- 객체 생성 가능

**인터페이스 Interface**
- 추상체 abstract : 기능 (method) 선언만 됨 ({} X)
- 객체 생성 불가 (추상 method)

**확장 Extends**
- 좌우 동일 타입 (하위 extends 상위)
- class extends class / interface extends interface1, interface2, ... interfaceN

**구현 Implements**
- class implements interface1, interface2, ... interfaceN
