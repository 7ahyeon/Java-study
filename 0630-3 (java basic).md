# Java-study

**실수형 float** 
- 4byte (범위 : 1.4E-45 ~ 3.4028235E38)
- 소수점 이하 데이터 저장 가능
- 표시 형식 : 숫자.숫자 + 접미사 f/F
- 단, 소수점 이하 6자리까지 정확도 보장 (연산 누적시 정확도 감소)

**실수형 double** 
- 8byte (범위 : 4.9E-324 ~ 1.7976931348623157E308)
- 소수점 이하 데이터 저장 가능
- 표시 형식 : 숫자.숫자, 숫자.숫자 + 접미사 d/D
- 단, 소수점 이하 6자리까지 정확도 보장 (연산 누적시 정확도 감소)
