- "홍길동", 100, 90, 81
- "이순신", 95, 88, 92
- "김유신", 90, 87, 77
2. ArrayList 타입의 변수(list)에 저장하고
3. list에 있는 데이터를 사용해서 전체 데이터 화면출력
- 성명    국어   영어  수학    총점     평균
- --------------------------
- 홍길동   100  90  81  270  90.33
4. 김유신 국어 점수를 95 점으로 수정 후 김유신 데이터만 출력
