# 반복문

**for문**
- for (초기값 ; 실행(종결)조건 ; 증감설정) {}
- 실행(종결)조건 true일 때 실행문 처리
- -for(; ;) {} : 무한 반복

```java
for (int i = 0; i < 5; i++) {
	System.out.println("*");
}
```
- 실행 순서 : int i = 0 -> i <5 -> System.out.println("*") -> i++
